
The 'Merge_Sanger_v3.py' is the Python program of this study.

The folder ‘compare_by_needle’ contains the merged sequence done by Merge_Sanger_v2.py, the merged sequence done by Merge_Sanger_v3.py, and the alignment result file produced by needle.

The folder ‘v2’ contains the seven example input files, the version 2 of the program, the improvements txt file note for the version 2 of the program, and the merged sequence done by Merge_Sanger_v2.py.

The folder ‘v3’ contains the example input files, the procedure files of the running of Merge_Sanger_v3.py (including running log), and the saved result file of Merge_Sanger_v3.py.
